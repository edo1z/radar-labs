# radar-labs

Small, self-contained experiments triggered by interesting items surfaced in
[daily-radar](https://github.com/edo1z/daily-radar). Each experiment is a folder
under this repo: a minimal MVP, run instructions, and what was learned.

Many experiments are tiny on purpose — the goal is to actually run new things
within a day or two of them landing, and write down what was easy, what was
not, and what the docs did not say.

## Layout

```
radar-labs/
├── README.md
├── <YYYY-MM-DD>-<slug>/
│   ├── README.md            # what / how to run / findings
│   ├── ...                  # code
│   └── media/               # screenshots, short demos
└── ...
```

Each experiment is independent. License per-experiment is MIT unless the README
under that folder says otherwise (some upstream weights or models have their
own terms — those are called out in the experiment's README).
